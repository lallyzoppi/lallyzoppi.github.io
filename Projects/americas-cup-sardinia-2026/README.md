# America’s Cup Cagliari 2026 Analytics
Inspired by the America’s Cup regatta conditions (Louis Vuitton 38th America’s Cup Preliminary Regatta), we propose a Sailing Performance Analytics Project Using Simulated Racing Telemetry. The goal is to reproduce typical high-performance sailing scenarios, including strong wind conditions, high-speed foiling, and tactical manoeuvres similar to those experienced in AC40 fleet racing.
Project for sailing data analysis using Python, Streamlit and Machine Learning.


## Run
pip install -r requirements.txt
streamlit run app/streamlit_app.py
